# 🛒 Blinkit Data Analysis — Capstone Project

> An end-to-end data analysis of Blinkit's grocery sales data using **Excel**, **SQL**, **Power BI**, and **Python** to uncover actionable business insights.

---

## 📌 Project Overview

This capstone project simulates a real-world data analyst workflow by performing a comprehensive analysis of Blinkit-style grocery sales data. The goal is to analyze sales performance, customer satisfaction, and inventory distribution across different outlet types and locations — and present findings through interactive dashboards and visualizations.

---

## 🎯 Objectives

- Analyze **total sales**, **average sales**, **number of items**, and **average customer ratings**
- Identify top-performing **product categories** and **outlet types**
- Understand the impact of **outlet size**, **location tier**, and **fat content** on sales
- Derive actionable **business recommendations** from data

---

## 🧰 Tools & Technologies

| Tool       | Purpose                                              |
|------------|------------------------------------------------------|
| **Excel**  | Data cleaning, pivot tables, and initial dashboards  |
| **SQL**    | Data extraction, KPI queries, joins, aggregations    |
| **Power BI** | Interactive dashboards, DAX measures, visual reports |
| **Python** | EDA, data manipulation (Pandas), visualization (Matplotlib, Seaborn) |

---

## 📊 Key KPIs Analyzed

- 💰 **Total Sales** — Overall revenue from all items sold
- 📦 **Number of Items** — Count of distinct products sold
- ⭐ **Average Rating** — Customer satisfaction score
- 📈 **Average Sales per Outlet** — Performance benchmarking

---

## 🔍 Key Insights

- Fruits, vegetables, and snack foods are the **top-selling categories**
- Consumers show a preference for **low-fat products**, indicating health-conscious buying behavior
- **Medium-sized outlets in Tier 3 locations** show the highest profitability
- Sales show seasonal fluctuations — opportunities exist for targeted promotions
